# Selectel IP Sniper Bot

Telegram-бот для автоматического поиска (сниффинга) плавающих IP-адресов (Floating IP) в облаке Selectel, которые попадают в заданный белый список подсетей. Позволяет управлять несколькими аккаунтами Selectel, отслеживать баланс, автоматически удалять «мусорные» IP и получать уведомления о найденных адресах.

## Возможности

- **Мультиаккаунтность** – одновременно работайте с несколькими проектами Selectel.
- **Автоматический перебор IP** – бот создаёт IP в указанных регионах, проверяет вхождение в whitelist и сразу удаляет неподходящие, экономя средства.
- **Гибкие настройки** для каждого аккаунта:
  - Активные регионы (ru-1, ru-2, …)
  - Размер батча (сколько IP создавать за раз)
  - Пауза между циклами
  - Порог баланса для автоматической остановки
  - «Умные зоны» – динамический выбор региона с бо́льшим свободным пулом
  - Автоисключение зон при высоком проценте повторов /24 подсетей
- **Мониторинг в реальном времени** – дашборд со статистикой, прогнозом остатка средств, списком найденных IP.
- **Управление через Telegram** – запуск/остановка, просмотр статуса, экспорт данных, перезагрузка whitelist.
- **Прокси-поддержка** – работа через SOCKS5-прокси для каждого аккаунта.
- **Безопасное завершение** – при остановке бот удаляет все временные IP и присылает итоговый отчёт.

## Как это работает

1. Вы загружаете белый список подсетей (файл `whitelist.txt`).
2. Добавляете аккаунты Selectel (токен и project_id) через Telegram.
3. Запускаете поиск для нужного аккаунта.
4. Бот в цикле:
   - выбирает регион,
   - создаёт N плавающих IP,
   - проверяет каждый IP на вхождение в whitelist,
   - если IP подходит – уведомляет админа и останавливает поиск (аккаунт переходит в режим ожидания),
   - если нет – сразу удаляет IP,
   - делает паузу, отправляет отчёт, отслеживает баланс.
5. При достижении порога баланса или исчерпании всех зон аккаунт автоматически останавливается.

## Требования

- Python 3.10+
- Аккаунт в [Selectel](https://selectel.ru/) с доступом к VPC
- Токен доступа к API Selectel (можно получить в панели управления)
- Telegram-бот (получить токен у [@BotFather](https://t.me/BotFather))
- Файл `whitelist.txt` с подсетями в формате CIDR

## Установка

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/rasnzz/selectel-ip-sniper-bot.git
   cd selectel-ip-sniper-bot
```

1. Создайте виртуальное окружение и активируйте его
   ```bash
   python -m venv venv
   source venv/bin/activate   # Linux/Mac
   venv\Scripts\activate      # Windows
   ```
2. Установите зависимости
   ```bash
   pip install -r requirements.txt
   ```
3. Создайте файл .env (скопируйте из примера)
   ```env
   TG_TOKEN=ваш_токен_бота
   ADMIN_ID=ваш_telegram_id
   TG_PROXY=socks5://user:pass@host:port   # опционально, если нужен прокси для бота
   ```
4. Создайте папку data и файл whitelist.txt
   ```bash
   mkdir data
   touch data/whitelist.txt
   ```
   Пример содержимого whitelist.txt:
   ```
   192.168.0.0/24 # Офисная сеть
   10.0.0.0/16
   172.16.0.0/12
   ```
   Поддерживаются комментарии после #. IP-адреса будут проверяться на принадлежность любой из перечисленных подсетей.

Запуск

```bash
python bot.py
```

Бот запустится, загрузит сохранённые аккаунты (если есть) и начнёт принимать команды в Telegram.

Использование бота

Доступ имеет только администратор (указанный ADMIN_ID).

Главное меню

После команды /start появляется клавиатура с двумя кнопками:

· 👥 Аккаунты – управление аккаунтами Selectel.
· 📋 Подсети – просмотр текущего белого списка и возможность его перезагрузки.

Управление аккаунтами

1. Добавить аккаунт
   Нажмите «Аккаунты» → «➕ Добавить аккаунт».
   Бот запросит:
   · Название (например, «Мой проект»)
   · SELECTEL_TOKEN (можно скопировать из панели Selectel)
   · PROJECT_ID (ID проекта в Selectel)
   · Прокси (можно пропустить) – формат socks5://user:pass@host:port
2. Панель аккаунта
   В списке аккаунтов нажмите на нужный. Откроется меню:
   · 🚀 Запустить / ⏸ Остановить – начать или приостановить поиск.
   · 📊 Статус – подробный дашборд: баланс, количество проверок, найденные


, прогноз, анализ по зонам и ближайший промах.
   · ⚙️ Настройки – изменение параметров:
     · Зоны (выбор регионов для перебора)
     · Пауза между циклами (от 1 до 10 минут)
     · Батч (сколько IP создавать за раз: 1,2,3,5)
     · Порог баланса (при достижении автоматическая остановка)
     · Умные зоны (вкл/выкл)
     · Авто-пауза зон (процент повторов /24 для исключения зоны)
   · 📥 Экспорт – скачать файл со всеми уникальными IP, проверенными этим аккаунтом.
   · 🌐 Прокси – изменить или удалить прокси.
   · 🗑 Удалить – удалить аккаунт (необратимо).
3. Запуск поиска
   После нажатия «Запустить» бот начнёт цикл создания и проверки IP. Каждый цикл (пауза) он присылает отчёт:
   · Какие IP были созданы (с пометкой повторений)
   · Ближайший IP к белой подсети (если есть)
   · Текущий баланс и прогноз оставшихся проверок
4. Найден IP из белого списка
   Бот немедленно присылает сообщение с IP, регионом, меткой из whitelist (если задана). Поиск для этого аккаунта останавливается. Чтобы продолжить, нужно снова нажать «Запустить» (бот удалит найденные IP и начнёт заново).

Управление белым списком

· Кнопка 📋 Подсети показывает все подсети из whitelist.txt.
· Кнопка 🔄 Перезагрузить whitelist – позволяет обновить список без перезапуска бота. Полезно, если вы добавили новые подсети в файл.

Формат whitelist.txt

· Каждая строка – подсеть в формате CIDR (например, 192.168.0.0/24).
· Можно добавить комментарий после #, например: 192.168.0.0/24 # Офисная сеть. Этот комментарий будет показываться в уведомлениях о найденном IP.
· Пустые строки и строки, начинающиеся с #, игнорируются.

Важные замечания

· Стоимость: Selectel тарифицирует каждый созданный Floating IP (даже если он сразу удалён). Настройте паузу и батч разумно, чтобы не тратить лишние средства.
· Rate limits: API Selectel имеет ограничения. Бот обрабатывает ошибку 429 (слишком много запросов) и делает паузу на 10 минут.
· Прокси для бота: если вы укажете TG_PROXY в .env, весь трафик Telegram-бота пойдёт через этот прокси. Для каждого аккаунта можно задать свой прокси (отдельно).
· Безопасность: файл data/accounts.json хранит токены Selectel в открытом виде. Убедитесь, что доступ к серверу ограничен.

Команды Telegram

· /start – показать главное меню.

(Все остальные действия – через кнопки.)

Остановка бота

Нажмите Ctrl+C в терминале. Бот корректно завершит работу:

· Остановит все активные аккаунты.
· Удалит все «мусорные» IP.
· Пришлёт итоговое сообщение со статистикой.

Устранение неполадок

· Бот не отвечает – проверьте, что TG_TOKEN и ADMIN_ID верны, и бот запущен.
· Ошибка авторизации (401) – неверный SELECTEL_TOKEN или PROJECT_ID. Проверьте их в панели Selectel.
· Whitelist не загружается – убедитесь, что файл data/whitelist.txt существует и имеет корректный формат.
· Прокси не работает – проверьте формат socks5://user:pass@host:port. Бот использует aiohttp-socks. IP
